# nikunj

A collection of small C practice programs.

## Programs

- `computer_sales.c` — reads the number of computers sold and their price, then prints the bonus, commission, and gross salary for a salesperson (base salary 1500, bonus of 200 per unit, 2% commission on sales).
- `home3.c` — a minimal printf example.

## Building

Compile any program with:

```sh
cc computer_sales.c -o computer_sales
./computer_sales
```

Compiled binaries (`a.out`, etc.) are ignored by git — see `.gitignore`.
